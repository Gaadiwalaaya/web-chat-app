# WebChat

A simple real-time chat application built as a practice project to learn **WebSockets**.

## Features
- Real-time messaging between users
- Simple authentication flow
- Message persistence

## Tech Stack
- **Backend**: Node.js, Express, Socket.io (WebSockets)
- **Frontend**: React, Vite
- **Database**: MongoDB

## What I Learned
This project was created to understand the fundamentals of real-time communication using WebSockets, specifically how to handle persistent connections and emit/listen for events between client and server.
